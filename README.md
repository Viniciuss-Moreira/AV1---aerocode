# Aerocode - AV1 - 2 DSM

## Como rodar o projeto

Certifique-se de ter o [Node.js](https://nodejs.org/) instalado em sua máquina.

1. Clone e abra a pasta do repositório
   ```bash
   git clone https://github.com/Viniciuss-Moreira/AV1---aerocode && cd AV1---aerocode && cd av1
   ```

2. Instale as dependências do projeto executando o comando abaixo na raiz do diretório:
   ```bash
   npm install
   ```

2. Para compilar e executar o código, utilize o comando de inicialização definido no projeto:
   ```bash
   npm start
   ```
